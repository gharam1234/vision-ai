# 위험구역 침입 비주얼 경고 UI 강화 계획 - 3차 계획서

## 프로젝트 개요

위험구역 침입 감지 시 사용자가 한눈에 인지할 수 있도록 OpenCV 디스플레이 화면의 경고 비주얼을 대폭 강화합니다.

---

## 강화된 비주얼 UI 사양

1. **전체 화면 빨간색 테두리 깜빡임 (0.3초 주기)**: 
   - 침입이 발생하고 있는 동안 화면 가장자리에 두꺼운 빨간색(두께 10px) 테두리가 깜빡여 긴급 상황임을 표현합니다.
2. **상단 중앙 대형 경고 배너**:
   - 기존 좌측 상단의 작고 투박한 배너 대신, 화면 **상단 중앙**에 반투명한 진한 빨간색 가로 배너를 배치합니다.
   - 배너 내부에 큰 글씨로 `[ ⚠️ EMERGENCY ALARM ]` 문구와 함께 `작업자#1 -> 위험구역 A (침입 중)` 정보를 교차 출력합니다.
3. **경고 발생 대상(작업자) 시각화 강화**:
   - 침입 중인 작업자의 바운딩박스 네 모서리에 조준선(Corner Bracket) 느낌의 강조 표시를 더합니다.
