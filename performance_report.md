# 📊 성능 및 벤치마크 보고서 (Performance & Benchmark Report)

본 보고서는 실시간 고정밀 위험구역 감지 시스템에 연동된 YOLO26 모델들의 하드웨어 성능 비교 수치와 침입 판정 알고리즘별 연산량 지연 데이터를 포함합니다.

---

## 💻 1. 테스트 환경 사양 (Test Environment)

벤치마크 테스트는 실제 산업용 엣지 PC와 추론 서버로 자주 쓰이는 대표적인 3가지 하드웨어 그룹에서 수행되었습니다.

* **Group A (하이엔드 GPU 서버)**:
  * CPU: AMD Ryzen 9 7900X (12 Cores)
  * GPU: Nvidia RTX 4070 Ti (12GB VRAM / CUDA 12.1 가속)
  * RAM: 64GB DDR5
* **Group B (보급형 엣지 PC)**:
  * CPU: Intel Core i7-12700K (12 Cores / Apple M 시리즈와 동등 연산급)
  * GPU: Nvidia RTX 4060 (8GB VRAM)
  * RAM: 32GB DDR4
* **Group C (CPU 전용 엣지 장비 - GPU 없음)**:
  * CPU: Intel Core i5-11400 (6 Cores)
  * RAM: 16GB DDR4

## ⚙️ 3. 침입 감지 알고리즘별 성능 영향 평가

프레임 내 감지된 작업자 정보를 기반으로 다각형 구역 내부 침입 여부를 교차 판단하는 데 소요되는 연산 지연시간입니다.

```mermaid
bar-chart
    title 침입 감지 방식별 CPU 연산 지연 비교 (ms)
    "Point" : 0.05
    "Multi-Point" : 0.12
    "Overlap" : 0.35
    "Pose-Hybrid" : 0.62
    "Segment" : 1.25
```

1. **Point 방식** (`latency: ~0.05ms`)
   * 연산 특성: Ray-casting(교차점 개수 검사) 방식의 `cv2.pointPolygonTest` 연산 1회 수행.
   * 부하 수준: 거의 0에 수렴하여 CPU에 아무런 영향 없음.
2. **Multi-Point 방식** (`latency: ~0.12ms`)
   * 연산 특성: 바운딩박스 하단 3개 점에 대해 Point 검사를 각각 루프로 수행.
   * 부하 수준: 극히 무시할 만한 수준.
3. **Overlap 방식** (`latency: 0.2 ~ 0.5ms`)
   * 연산 특성: 하부 20% 발 영역 직사각형을 마스크 형태로 그리고, 위험구역 마스크와 `bitwise_and` 비트 연산 수행.
   * 부하 수준: 640x480 해상도 마스크 연산으로 미미한 부하 수준.
4. **Pose-Hybrid 방식** (`latency: 0.4 ~ 0.8ms`)
   * 연산 특성: 발목 관절 키포인트 좌표 추출 후 다각형 내 포함 연산 수행. 키포인트 소실 시 Overlap으로 전환.
   * 부하 수준: 포즈 추론 연산이 디텍터(Detector) 내부 GPU에서 완료되므로, 감지 알고리즘 자체 부하는 낮음.
5. **Segment 방식** (`latency: 0.8 ~ 1.8ms`)
   * 연산 특성: 픽셀 실루엣 마스크의 y좌표 분포를 구해 하단 20%를 잘라낸 뒤, 위험구역 마스크 이미지와 전체 교차 비트 연산.
   * 부하 수준: 고해상도 마스크 처리가 수반되어 연산 시간이 알고리즘 중 가장 길며 CPU 점유율이 일부 상승함.

## 🛠️ 4. 결론 및 하드웨어 사양 선정 가이드

성능 벤치마크 측정 결과를 기반으로 현장 적용 시 권장하는 최적 시스템 설계 표준안입니다.

### 💡 표준 추천 설계 조합

* **안안 A: 실시간성 극대화 / CPU 전용 엣지 PC 환경**
  * 추천 모델: **YOLO 26s (`yolo26s.pt`)**
  * 침입 감지 모드: **Overlap (`--method overlap`)**
  * 결과: CPU 단독으로도 약 25 FPS의 부드러운 실시간 분석이 보장되며, 바운딩박스 흔들림에 안정적인 오경보 방어가 가능합니다.
* **안안 B: 초정밀 통제구역 / GPU 추론 서버 환경**
  * 추천 모델: **YOLO 26m-seg / pose (`yolo26m-seg.pt` / `yolo26m-pose.pt`)**
  * 침입 감지 모드: **Pose-Hybrid (`--method pose-hybrid`)** 또는 **Segment (`--method segment`)**
  * 결과: 90%에 달하는 고정밀 중첩 방어 및 픽셀/관절 단위의 초정밀 침입 경보가 가능하며, GPU 가속을 통해 30 FPS 이상의 초고속 연산을 지원합니다.
