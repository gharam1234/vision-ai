# 실시간 위험구역 마우스 드로잉 설정 기능 구현 계획 - 2차 계획서

## 프로젝트 개요

실시간 영상 감지를 시작하기 전에, 사용자가 **마우스 클릭으로 위험구역 다각형을 직접 그리고** 설정할 수 있도록 OpenCV 기반 대화형 드로잉 기능을 추가합니다.

---

## 드로잉 동작 방식

1. 프로그램 실행 시 첫 프레임에서 일시정지 상태로 대기합니다.
2. **마우스 왼쪽 클릭**: 다각형의 꼭짓점 추가
3. **마우스 오른쪽 클릭**: 직전에 그린 꼭짓점 하나 취소 (Undo)
4. **키보드 'c'**: 현재 그리고 있는 다각형 전체 초기화 (Clear)
5. **키보드 'Enter' 또는 'Space'**: 현재 다각형을 완성하고 저장 (여러 구역 그리기 가능)
6. **키보드 's'**: 설정 완료하고 실시간 영상 처리/위험 감지 시작
7. **키보드 'q'**: 프로그램 종료
